
#

Este é um space shooter feito usando P5.js que eu fiz para o trabalho final de FUP.

# Controles

- Mouse: Move a nave
- Clique esquerdo: Atira
- Clique direito: Ataque especial `(Precisa que a barra esteja cheia)`
- Backspace: Pausa

# Cheats

- Debug mode: Tecla D
- God mode: Tecla G durante o debug
